# Online_Examination_System
📌 Project Overview

The Online Examination System is a desktop-based application developed using Java Swing, JDBC, and MySQL. It provides a secure and efficient platform for conducting online multiple-choice examinations. The system automates examination management, question delivery, answer evaluation, and result generation, reducing manual effort and improving accuracy.

🎯 Objectives
Automate the examination process.
Eliminate manual evaluation.
Provide instant result generation.
Ensure secure student authentication.
Improve efficiency and accuracy in conducting exams.
🚀 Features
Student Module
Secure student login and authentication.
View available examinations.
Attend MCQ-based online tests.
Countdown timer for each examination.
Automatic submission when time expires.
Instant score calculation.
Result and performance display.
Admin Module
Secure admin login.
Add, update, and delete questions.
Manage student accounts.
Create and manage examinations.
Set examination duration.
View student results and reports.
Maintain question database.
Examination Features
Multiple Choice Questions (MCQs).
One correct answer per question.
Random question generation.
Automatic answer evaluation.
Real-time score calculation.
Instant result generation after submission.
